## Descripción del Proyecto
Este proyecto integra la investigación teórica de diversas estructuras arbóreas y la implementación técnica de un **Árbol Binario de Búsqueda (BST)** en Java. Se enfoca en demostrar la eficiencia algorítmica y el manejo de estructuras dinámicas.

---

## Fase 1: Fichas Técnicas
Se incluye un análisis de las siguientes estructuras en la documentación del proyecto:
1.  **BST (Binary Search Tree)**: Base de las búsquedas binarias.
2.  **AVL**: Árbol auto-balanceado por altura.
3.  **Árbol B/B+**: Optimización para bases de datos.
4.  **Trie**: Búsqueda por prefijos (diccionarios).
5.  **Heap**: Gestión de prioridades.

---

##  Fase 2: Implementación en Java (BST)
Se ha desarrollado un **Árbol Binario de Búsqueda** desde cero con las siguientes funcionalidades:

###  Características Implementadas:
*    **Inserción:** Lógica recursiva eficiente.
*    **Búsqueda:** Retorno booleano instantáneo.
*    **Eliminación:** Manejo de casos complejos (hojas, 1 hijo, 2 hijos).
*    **Recorridos:** In-Order (ordenado), Pre-Order y Post-Order.

###  Análisis de Complejidad:
| Operación | Caso Promedio | Peor Caso |
| :--- | :--- | :--- |
| **Búsqueda** | O(log n) | O(n) |
| **Inserción** | O(log n) | O(n) |
| **Borrado** | O(log n) | O(n) |
